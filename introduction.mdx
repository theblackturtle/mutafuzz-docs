## Overview

MutaFuzz is a powerful Burp Suite extension that brings **Python scripting** to HTTP fuzzing. Built on the Montoya API, it provides a flexible, programmable fuzzing framework perfect for bug bounty hunting, penetration testing, and security research.

## Key Features

### Python-Powered Fuzzing

Unlike traditional fuzzing tools with limited scripting capabilities, MutaFuzz gives you the full power of Python:

- **Fluent API**: Chain methods for readable, maintainable scripts
- **Response Callbacks**: Process results with custom Python functions
- **Decorators**: Stack filters to build complex logic (`@filter.status([200])`)
- **Full Montoya Access**: Direct access to Burp's complete API

```python
@filter.status([200])
@filter.interesting()
def handle_response(req):
    if "admin" in req.text:
        table.add(req)
```

### Intelligent Learn Mode

Learn mode analyzes baseline responses to automatically filter out noise:

1. **Calibration Phase**: Send random payloads to establish baselines
2. **Pattern Detection**: Identify response variations by status, length, body hash
3. **Smart Filtering**: Only surface responses that differ from learned patterns
4. **Group-Based**: Multiple learn groups for different payload types

<Tip>
  Learn mode drastically reduces false positives when fuzzing for hidden
  endpoints, parameters, or vulnerabilities.
</Tip>

## Use Cases

### Bug Bounty Hunting

- Fuzz for hidden endpoints with learn mode filtering
- Test parameter injection with custom transformations
- Chain requests for multi-step exploitation
- Session-aware testing with `session.set()` API

### Penetration Testing

- Automate repetitive fuzzing tasks with Python
- Custom authentication flows via scripting
- Bypass WAF detection with randomized payloads
- Generate comprehensive reports from results table

### Security Research

- Prototype new fuzzing techniques quickly
- Analyze response patterns with full API access
- Test complex scenarios requiring stateful logic
- Export results for further analysis

## Quick Example

Here's a complete fuzzing script that demonstrates MutaFuzz's power:

```python
@filter.status([200])
@filter.interesting()
def handle_response(req):
    """Only show 200 OK responses marked interesting by learn mode"""
    table.add(req)

def queue_tasks():
    # Calibration: establish baseline responses
    for i in range(3):
        random_path = utils.randstr(length=8)
        fuzz.payloads([random_path]).learn_group(1).queue()

    # Fuzz with wordlist
    for word in payloads.wordlist(1):
        fuzz.payloads([word]).queue()

    fuzz.done()
```

<Note>
  Learn mode (`.learn_group(1)`) automatically filters responses, showing only
  those that differ from calibration baselines.
</Note>
